# Rezort - Hotel Reservation Website

**Rezort** is a responsive hotel reservation website that allows users to browse rooms, check availability, and make bookings online. The project focuses on creating a user-friendly interface using modern front-end technologies.

## Features
- Browse hotel rooms and details
- Check room availability
- Make reservations online
- Responsive design for mobile and desktop
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/M0habAOT/Rezort.git
   ```
2. Open `index.html` in your web browser.
3. Enjoy exploring the website locally.
